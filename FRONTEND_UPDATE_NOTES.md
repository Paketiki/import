## Инструкции по обновлению CSS

1. Откройте репозиторий в VS Code
2. Найдите файл `static/styles/style.css`
3. Скопируйте содержимое вашего нового `style.css`
4. Замените всё содержимое файла
5. Сохраните (Ctrl+S)
6. Git автоматически отметит изменение
7. Сделайте commit:
   ```bash
   git add static/styles/style.css
   git commit -m "Update: Replace CSS with new version from user"
   git push
   ```

## Тестирование

После обновления:
1. Запустите приложение: `uvicorn main:app --reload`
2. Откройте http://localhost:8000
3. Проверьте:
   - [ ] Все кнопки работают правильно
   - [ ] Фильтры функционируют (поиск, жанры, оценки)

## Возможные проблемы и решения

### Если кнопки не работают
- Проверьте, что пути в HTML правильные (`/static/styles/style.css`, `/static/js/script.js`)
- Убедитесь, что FastAPI сервер запущен на `http://localhost:8000`

### Если CSS не применяется
- Очистите кэш браузера (Ctrl+Shift+Delete)
- Проверьте консоль браузера (F12) на ошибки загрузки файлов
- Убедитесь, что файл `static/styles/style.css` существует

### Если API не отвечает
- Проверьте логи FastAPI сервера
- Убедитесь, что все зависимости установлены (`pip install -r requirements.txt`)
- Проверьте, что база данных `movies.db` существует
